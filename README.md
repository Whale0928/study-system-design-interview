### 진행 방식

- 각 일정마다 1주제 단위로 발표

- 챕터의 규모에 따라 1장에 대한 발표를 여러명이 나누어 진행
- 각 주제에 대해 PR 생성 후 발표자 외에도 코멘트로 참여
